# Assets folder

Copy all your existing portfolio assets here from the original project's `assests` folder so that images, videos, and files (e.g. `affan1.jpg`, `bg.png`, `icon.png`, `ProVideo/*.mp4`, CV, certificates, etc.) are available at `/assests/...`.

If your assets are already in the project root as `assests`, copy that folder into `public/` so the final path is `public/assests/`.

**Resume button:** Place your resume PDF as `Resume.pdf` in this folder (`public/assests/Resume.pdf`) so the Home page "Resume" button opens it correctly.
